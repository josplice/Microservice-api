# Devcamper Api

> Backend API for DevCamper application which is a bootcamp directory website

## Usage

rename " config/config.env.env" to "config/config.env" and update the values to your own

## Install dependacies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in production mode
npm start
```
